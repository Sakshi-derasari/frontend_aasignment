Explain in your own words (3-4 lines) how React's Virtual DOM makes updating the UI faster compared to traditional DOM manipulation.

Ans. React uses a Virtual DOM, which is a lightweight copy of the real browser DOM. When something changes, React first updates this Virtual DOM and compares it with the previous version (diffing). It then updates only the specific parts of the real DOM that changed, instead of reloading everything. This selective updating makes the UI much faster and more efficient than traditional DOM manipulation.
